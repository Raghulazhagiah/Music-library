                                                    🎵 Music Library Management API
The Music Library Management API is a robust system designed to help organizations manage their collection of Artists, Tracks, and Albums efficiently. Each organization has a single Admin responsible for overseeing the system and its users.

🚀 Features
Artist, Album & Track Management: Create, update, and organize your music collection.

User Roles: Admins manage users, while regular users can explore and personalize their experience.

Favorites System: Mark favorite artists, albums, and tracks for quick access.

Personalized Experience: Users can curate their library for easy discovery.

This API serves as a foundation for building scalable music library solutions within organizations.

📌 Tech Stack: Node.js, Express, PostgreSQL
